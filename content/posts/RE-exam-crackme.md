# Reversing the malware

During our OS Security course, we learned about the basics of reverse engineering. At the end of the course, we were given a crackme to solve as final exam.

The crackme is a Windows executable employing malware techniques. It wasn't expected from us to fully reverse the malware but at least analyze it, understand what's going on and eventually find the flag.

The crackme binary can be downloaded [here](/exam_crackme.exe) and my analysis (in French) can be found [here](/report-exam_crackme.pdf).
